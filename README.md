Файл исходных кодов для прошивки микроконтроллера stm32h755ZIQ.

Программировался для отладочной платы NUCLEO-H755ZI-Q.

Здесь развёрнут веб сервер. По умолчанию устанавливается адресс 192.168.1.129.
<img width="2512" height="1317" alt="image" src="https://github.com/user-attachments/assets/63412f92-029c-4d53-a7e8-739ac53c4b30" />

Сервер пока что только демонстративный, кнопочки не тыкаются, но уже сейчас есть данные с ацп. 
В папке CM7\Middlewares\mongoose есть html файл scope, из которого с помощью xxd сделан 
web_content.h, в котором уже чистые данные для отправки. 

Пока не реализовано, но есть намёки, что скорее всего работает:
- межъядерное взаимодействие между ядром cm4 и cm7;
- На сайте есть заголовки ацп цап и ШИМ. Планируется считывать данные с такой переферии
и через механим межъядерки гнать данные с м4 на м7, и далее по ethernet (на NUCLEO-H755ZI-Q
он благо сделан) гнать на веб-сервер.
